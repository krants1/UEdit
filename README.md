
### run/install
* create DB: 
  * install PostgreSql 
  * create database "test"
  * change DBConfig in [./server/main.ts](https://github.com/krants1/UEdit/blob/master/server/main.ts)
* run server:
  * cd server
  * npm install
  * npm run start
* run client:
  * cd..
  * npm install
  * npm run start
